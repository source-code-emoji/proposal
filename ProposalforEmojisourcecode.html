3. Factors for Inclusion

a. Expresses multiple concepts

The primary concept of the emoji is source code: the human-readable text-based set of instructions that make up the blueprints for computer programs, often referred to as applications or simply "apps". Therefore the emoji could also be used to express the concepts of computer programs, apps, instructions, rules, algorithms, or software in general.

b. Can be used with other emoji to convey additional concepts

There are many concepts related to source code that can be conveyed when combined with other emoji:

  1. 📦: Software package
  2. ✅: Code change approved
  3. 💩 / ☠️ : Code smell or bad code
  4. 💸: Tech debt
  5. 🐛: Bug in code
  6. 🔥: Code that caused a major incident
  7. 🌍 or 🤝 : Open source
  8. 🔐: Closed source
  9. 🤖 or ✨ or 🪄  or 💬: AI prompt

And the emoji could be used in sentences:

  1. 👀 🙏? : Could you review my code please?
  2. Did you 👁️ the 💦? : Did you see the leaked source code?
  3. 📅 ⏩💸 : Today’s code will be tomorrow’s tech debt.
  4. If you want trust show the
  5. My  is powered by ☕!

c. Breaks new ground

There are currently no emoji that represent source code. The compact disk 💿 or floppy disk 💾 were used for representing software in general, but a) they are outdated media and b) are not specific enough: source code is related to the creation of software, much like bricks 🧱 are part of the creation of a house 🏠.

d. Is legible and visually distinctive

The </> symbol is a widely used symbol for source code. While it appears to have been inspired by HTML, which uses the 3 characters </> frequently, HTML never actually uses them exactly in that order, and is not limited to describing just code in that language.

For example searching for "source code symbol" returns many icons that feature </> prominentally.


The foundation of the symbol is 5 straight line segments, and so there should not be a problem making this legible even in small sizes.

It is also visually similar to the symbol often used for (AI) prompts ">_", and so this emoji would be reasonable way of referring to AI prompts, especially when combined with 🤖 or ✨, which are emoji frequently used when discussing AI

4. Factors for Exclusion

a. Already represented

The closest existing icons are technologist and variations (🧑‍💻 technologist, 👨‍💻 man technologist and 👩‍💻 woman technologist). They do not represent the concept itself but only a person working with a computer. This could be a person writing source code, but this is separate from the source code concept itself. In fact, to represent a person writing source code technologist could be combined with the source code emoji 🧑‍💻.

As mentioned earlier, the compact disk 💿 or floppy disk 💾 were used for representing software in general, but a) they are outdated media and b) are not specific enough: source code is related to the creation of software, much like bricks 🧱 are part of the creation of a house 🏠.

b. Overly specific        

There are thousands of languages that "source code" can be written in. While strictly speaking  does resemble HTML, as mentioned earlier it is the general image to represent source code of any language, and it is even not completely accurate to represent HTML because the three characters </> don't appear in the language in that order with no characters between them.

c. Open-ended

The icon is inspired by HTML syntax, and theoretically there are a limited number of other syntax types that could attempt to be represented. For example C/C++/Java/JavaScript, also popular programming languages, use the characters { and } prominently in their syntax, and so a future emoji could be based on those characters. However the earlier image search for "source code symbol" shows that ''</>' appears to have won as the generic symbol for source code.

Also, HTML has its own logo, as do various other languages: often they are logotypes with the name of their language. This emoji is distinct from these and so not used to refer to any specific language, even HTML, and so should not result in the need to add an emoji for each.

d. Transient

Although source code is written in programming languages, and in general specific programming languages rise and fall in popularity, the source code concept itself remains the same. Every personal computer or phone uses literally billions of lines of source code to function, and so in fact source code is how emoji themselves are processed - source code is no more transient than emoji.

Although some languages fall in popularity, HTML, the language that inspired the  image, is the main language on the internet: every web page uses it, and there are no signs that this will change. And although it is currently argued that AI is changing how source code is written, the source code concept itself remains, and the source code emoji can aid representation of such code when combined with existing emoji such as 🤖 or ✨.

It is also decidedly less transient than the closest alternatives of compact disk 💿 or floppy disk 💾 emoji which are already outdated.

e. Justified by an existing emoji

The proposal is not justified by the existence existing emoji, other than how compact disk 💿 or floppy disk 💾 emoji are outdated.

Other information

The proposed design uses the </> symbol to represent source code. One of the possible colour palette choices was inspired by the historical green of monochrome monitors, but it is not required as shown by the earlier image search for "source code symbol".

This could however be substituted with a more abstract representation of code inspired by The Matrix franchise of films (1999 - 2021), which itself was partially inspired by the monochrome green.


However, we suspect such a choice may be overly transient and reliant on knowledge of the films.
